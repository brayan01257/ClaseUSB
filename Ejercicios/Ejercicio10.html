Alertas de Información sobre Predios

Las alertas se crean con la clase .alert, seguida de una clase de color contextual:

Éxito! Esta alerta indica que la información del predio se ha actualizado correctamente.
Información! Esta alerta proporciona información importante sobre nuevos predios disponibles.
Advertencia! Esta alerta indica que el predio puede tener problemas que necesitan atención.
Peligro! Esta alerta señala un problema crítico con el predio que debe ser atendido de inmediato.
Principal! Esta alerta destaca una acción importante relacionada con los predios.
Secundario! Esta alerta indica información adicional menos urgente sobre los predios.
Oscuro! Alerta de color gris oscuro para información menos prominente sobre predios.
Claro! Alerta de color gris claro para mensajes menos destacados.
